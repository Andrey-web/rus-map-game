## Игра «Карта России»

Интерактивная браузерная мини‑игра: перед вами карта России с отмеченными регионами. При клике по маркеру открывается стильное модальное окно с описанием места и одновременно проигрывается небольшой музыкальный аккорд, уникальный для каждого региона.

### Основные возможности

- **Интерактивные маркеры на карте**: кликайте по анимированным «огонькам» на карте, чтобы узнать больше о:
  - Санкт‑Петербурге (2 точки),
  - Москве,
  - Владимире,
  - Татарстане (Казань),
  - Кавказе,
  - Урале,
  - Сибири,
  - Дальнем Востоке.
- **Информационные карточки**: для каждого региона открывается модальное окно с:
  - названием и кратким подзаголовком,
  - текстовым описанием,
  - тематическим эмодзи,
  - небольшим хэштегом.
- **Звуковое сопровождение**: при открытии карточки синтезируется аккорд с помощью Web Audio API — у каждого региона свой «звуковой мотив».
- **Режим разработчика (поиск координат)**: можно быстро подбирать координаты новых точек на карте.

### Как играть

1. **Откройте игру**:
   - Просто откройте файл `index.html` в современном браузере (Chrome, Edge, Firefox, Safari).
   - Убедитесь, что рядом с ним лежит файл `rusmap.png` (карта России).
2. **Изучайте карту**:
   - Наводите курсор на маркеры, чтобы увидеть подпись региона.
   - Кликайте по маркеру — появится модальное окно с описанием и зазвучит аккорд.
   - Закрыть окно можно по нажатию на крестик, по клику по затемнённой области вокруг или клавишей `Esc`.
3. **Используйте подсказку координат (для добавления новых точек)**:
   - Зажмите `Shift` и кликните по любой точке карты.
   - В правом верхнем углу карты появится подпись вида `x: 18.5, y: 50.0` — это процентные координаты точки, которые можно добавить в массив `regions` в `index.html`.

### Требования

- Современный браузер с поддержкой:
  - `flexbox` и CSS‑анимаций,
  - свойства `object-fit: contain` для изображений,
  - Web Audio API (`AudioContext`).
- Включённый звук в системе и в браузере (для прослушивания аккордов).

### Структура проекта

- **`index.html`** — основная страница с разметкой, стилями и логикой игры (маркерные точки, модальные окна, звук).
- **`rusmap.png`** — изображение карты России, на котором располагаются все маркеры.

### Как добавить новый регион

1. Откройте `index.html`.
2. Воспользуйтесь режимом подсказки координат (Shift + клик по карте) и запишите `x` и `y`.
3. В массиве `regions` добавьте новый объект:

```js
{
  id: 'my-region-id',
  label: 'Название на подсказке',
  title: 'Заголовок в модалке',
  subtitle: 'Подзаголовок',
  desc: 'Описание региона.',
  emoji: '📍',
  tag: '#Хэштег',
  x: 25.0, y: 40.0,
  freq: [440, 554, 659] // частоты нот аккорда
}
```

4. Сохраните файл и перезагрузите страницу в браузере — новый маркер появится на карте.
